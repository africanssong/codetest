###  진행방식 
- 매주 1-2문제씩 풀이
- 주차별 폴더에 업로드
- 작성한 코드에 주석 달기
- 문제유형(DP, 그리디... 등)을 숙지 후 문제 푸는 것을 권장:exclamation:
- 매주 월요일 문제 확인. 이후 금요일까지 문제 풀이 진행 
  - 금요일까지 실패 시 온라인 검색을 통해 정답을 찾고 해당 코드의 주요 아이디어를 주석으로 표기.
  - 오답노트는 토-일 진행하여 월요일 이전까지 파일 업로드
